> # Installation

## Local installation

Make sure you have node installed on your device

Run the following code to clone the repository and install all required dependencies

```bash
$ git clone https://github.com/itzzzme/spotify-api.git
$ cd spotify-api
$ npm install
```

Start the server

```bash
$ npm start
```
